HireFlow is a full-stack **Applicant Tracking System (ATS)** built using the **MERN stack**.  
It streamlines the hiring process by managing candidates, interviews, and feedback with **role-based access control** for Admins, Recruiters, and Candidates.



## Features

###  Authentication & Authorization
- JWT-based authentication
- Role-based access control (RBAC)
- Roles supported:
  - Admin
  - Recruiter
  - Candidate

###  Candidate Management
- Create and manage candidate profiles
- Upload and track resumes
- Candidate status tracking (Applied, Shortlisted, Interviewed, Hired, Rejected)

###  Interview Management
- Schedule interviews
- Assign recruiters/interviewers
- Track interview status

###  Feedback System
- Interview feedback submission
- Ratings and comments
- Hiring decision tracking

###  Architecture
- Repository Pattern
- Service Layer
- DTO-based validation
- Dependency Injection (Inversify)
- Clean and scalable folder structure
